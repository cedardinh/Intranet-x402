# x402 Workspace (Organized View)

## Components
- `components/blockchain`: local chain token contract, deploy script, and stop script
- `components/facilitator`: x402 facilitator service
- `components/resource-server`: paywalled express resource server
- `components/mcp-bridge`: MCP bridge client service

## Runtime
- `x402-workspace/runtime/blockchain`: runtime artifacts generated by the flow

## Config
- `config/mcp.json`: MCP server config (linked to `.vscode/mcp.json`)

## Docs
- `docs/Operation Steps.md`: step-by-step runbook
- `docs/CODE_FLOW_GUIDE.md`: flow-level guide
- `docs/read.md`: project overview

## Compatibility
Root-level `Operation Steps.md`, `CODE_FLOW_GUIDE.md`, and `read.md` are symlinks for quick access.
